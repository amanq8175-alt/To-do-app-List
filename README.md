# 📝 To Do List App

A simple and responsive **To Do List Web App** built with **HTML, CSS, and JavaScript**.
It allows users to quickly add and remove tasks to stay organized.

## 🚀 Features

* Add tasks
* Delete tasks
* Clean and responsive UI

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
